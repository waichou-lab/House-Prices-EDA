# House Prices EDA Project

## Project Overview
Exploratory Data Analysis for Kaggle's House Prices: Advanced Regression Techniques competition.

## Dataset
- **Source**: [Kaggle House Prices Competition](https://www.kaggle.com/c/house-prices-advanced-regression-techniques)
- **Training Set**: 1,460 samples, 81 features
- **Test Set**: 1,459 samples, 80 features
- **Target Variable**: SalePrice

## Project Structure
- `notebooks/`: Jupyter notebooks for analysis
- `src/`: Python modules with reusable functions
- `data/`: Dataset files
- `reports/`: Generated analysis reports and visualizations
